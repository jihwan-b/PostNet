# PostNet: 연세대학교 고등교육혁신원 2025 윈터 워크스테이션 project

**PostNet**은 연세대학교 교내의 무분별한 게시판 정보를 해결하기 위한 **맞춤형 정보 큐레이션 웹 플랫폼**입니다.
쏟아지는 포스터 홍수 속에서 학생들의 관심사에 맞는 정보(동아리, 공모전, 채용 등)만을 필터링하여 제공합니다.



## 🛠 Tech Stack

* **Frontend:** React, Vite
* **Database:** Firebase (Firestore)
* **Deployment:** Vercel

## ✨ Key Features

1.  **카테고리 필터링:** 동아리, 공모전, 세미나 등 원하는 태그만 선택해서 보기
2.  **관심도 측정:** 방문자 수 및 클릭 데이터 집계 (Pre-Launch 검증용)
3.  **사용자 피드백:** 서비스에 대한 의견 및 별점 남기기
4.  **반응형 디자인:** 모바일과 PC 환경 모두 지원
